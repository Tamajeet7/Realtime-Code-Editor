# Realtime Code Editor

> A collaborative coding platform supporting real-time multi-user editing.

## Features

- JWT Authentication
- Real-time Collaboration
- Monaco Editor
- Room Management
- Live Cursor Tracking
- Multiple Programming Languages

---

## Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS

### Backend

- Express.js
- Socket.io
- Prisma
- PostgreSQL
